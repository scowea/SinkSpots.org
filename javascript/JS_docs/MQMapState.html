Class MQMapState

Object
   |
   +--MQObject

class MQMapState
extends MQObject


Defines the characteristics of a map. It includes such information as the map name, the coverage name, the latitude/longitude of the map Center, the map scale, and the image size.
See:

    - MQLatLng
    - MQPoint
Defined in mqobjects.js
